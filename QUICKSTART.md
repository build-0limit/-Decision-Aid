# 快速启动指南

## 📦 安装

```bash
# 安装依赖
npm install
```

## 🚀 运行

```bash
# 启动开发服务器
npm run dev
```

浏览器会自动打开 `http://localhost:5173`

## 🎮 使用演示

### 0. 配置 API（可选）
首次使用时，点击右上角的 **⚙️ 配置** 按钮：
- 选择"演示模式"可以直接体验（无需 API）
- 或配置你的 LLM API（OpenAI、Anthropic 等）
- 详细配置说明请查看 [CONFIG_GUIDE.md](CONFIG_GUIDE.md)

### 1. 输入决策问题
在首页输入框中输入你的问题，例如：
- "我应该选择哪个城市工作？北京、上海还是深圳？"
- "我应该买什么电子产品？"
- "我应该如何规划职业发展？"

### 2. 查看决策树
点击"生成决策树"按钮后，你会看到：
- 顶部的进度条
- 中间的树形可视化图
- 底部的当前问题和选项

### 3. 进行决策
- 树形图中，当前节点会有**紫色脉冲动画**
- 已访问的路径会**高亮显示**，连接线有**流动动画**
- 点击选项按钮进行选择
- 树会实时更新，动画展示你的移动过程

### 4. 查看结果
完成所有选择后：
- 看到最终的决策建议
- 查看完整的决策路径
- 树形图显示你走过的完整路径

## 🎨 视觉效果说明

### 节点图标含义
- ❓ 未访问的节点
- ✓ 已访问的节点
- 👉 当前所在节点（有脉冲动画）
- 🎯 最终结果节点

### 连接线颜色
- 灰色：未访问的路径
- 深灰色：已访问的路径
- 紫色（流动）：当前激活的路径

### 动画效果
- **脉冲动画**：当前节点会持续放大缩小
- **图标跳动**：当前节点的图标会上下跳动
- **流动动画**：激活路径的虚线会流动
- **悬停效果**：鼠标悬停时节点会放大

## 🔧 自定义配置

### 使用配置中心
1. 点击右上角"⚙️ 配置"按钮
2. 选择 LLM 服务商
3. 输入 API Key
4. 测试连接
5. 保存配置

详细说明请查看 [CONFIG_GUIDE.md](CONFIG_GUIDE.md)

### 修改决策树逻辑
编辑 `src/services/llm.js` 中的 `generateMockDecisionTree` 函数

### 调整视觉样式
- 节点样式：`src/components/DecisionTreeVisualization.vue`
- 界面样式：`src/App.vue`
- 全局样式：`src/style.css`

### 接入真实LLM API
1. 打开配置中心
2. 选择你的 LLM 服务商（OpenAI、Anthropic 或自定义）
3. 输入 API Key
4. 选择模型和参数
5. 测试连接并保存

无需修改代码！

## 📱 移动端使用

项目完全支持移动端：
- 响应式布局自动适配
- 触摸操作支持
- 横向滚动查看大型树结构

## 🐛 常见问题

### Q: 树形图显示不完整？
A: 尝试横向滚动，或者调整浏览器窗口大小

### Q: 动画不流畅？
A: 检查浏览器性能，关闭其他占用资源的标签页

### Q: 如何重新开始？
A: 在结果页面点击"开始新的决策"按钮
